<h2>DC pCategories &ndash; Prestashop Homepage Categories Module</h2>
<p><strong>DC pCategories</strong> is a free Prestashop module designed to display selected product categories directly on the homepage in a clean and customizable grid layout.</p>
<p>The module helps store owners highlight the most important sections of their catalog, making it easier for customers to navigate the store and quickly find the products they are looking for. By showcasing main categories on the homepage, stores can improve user experience, simplify navigation, and guide visitors directly to key product sections.</p>
<hr />
<h2>Key Features</h2>
<h3>Select categories to display</h3>
<p>Choose any categories from your Prestashop store and display them on the homepage as clickable category tiles.</p>
<h3>Drag &amp; drop category sorting</h3>
<p>Easily reorder categories using a drag &amp; drop interface in the admin panel.</p>
<h3>Configurable category grid</h3>
<p>Display categories in a flexible grid layout with configurable number of columns.</p>
<h3>Category images support</h3>
<p>Optionally display category images to make the section more visually appealing.</p>
<h3>Responsive layout</h3>
<p>The grid automatically adapts to different screen sizes, ensuring a smooth experience on desktop, tablet, and mobile devices.</p>
<h3>Multilingual support</h3>
<p>The module supports Prestashop's multilingual system, allowing different intro texts for each language.</p>
<hr />
